

# MindPilot Landing Page

A conversion-focused landing page for MindPilot — an AI-powered daily planning tool.

## Design Direction
- Dark gradient background with purple/blue AI-themed accents
- Modern, clean typography with high contrast
- Fully responsive mobile-first design
- Smooth scroll navigation

## Page Sections

### 1. Hero Section
- Bold headline: "Your AI That Plans Your Day Automatically"
- Subtext about reducing overwhelm and gaining focus
- Primary CTA button scrolling to email signup
- Gradient visual/abstract background element

### 2. Why MindPilot
- 4 feature cards with icons: auto-prioritization, smart scheduling, adaptive planning, motivation
- Contrast messaging: "Most task apps store to-dos. MindPilot thinks for you."

### 3. How It Works
- 4-step numbered visual flow with icons
- Tell goals → AI plans → Focused sessions → Track progress

### 4. Built For
- Grid of audience pain points with icons
- Overwhelmed, priorities, structure, real productivity

### 5. Early Access Signup
- Email input + "Get Early Access" button
- Bonus callouts: free premium, product influence, priority access
- Toast confirmation on submit (no backend yet — can add Supabase later to store emails)

### 6. Footer
- MindPilot branding and copyright
